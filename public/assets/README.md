# Imagens do site (Netlify / build)

Coloque aqui os arquivos de imagem para o site. **Eles precisam estar no repositório** para o Netlify encontrá-los no deploy.

- **`i.png`** – favicon (ícone da aba do navegador), ex.: 32x32 ou 48x48 px  
- **`Logo-Aberta.png`** – logo usada na sidebar e no menu mobile  

Depois de adicionar os arquivos, faça commit e push para o deploy no Netlify atualizar.
